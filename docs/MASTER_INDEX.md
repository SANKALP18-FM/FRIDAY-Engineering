## Project Vision

Build **FRIDAY**, a modular AI Operating System for Windows that combines natural conversation, intelligent planning, desktop automation, browser automation, memory, and specialized AI agents while remaining lightweight enough to run efficiently on a personal laptop.

# Engineering Rules

1. Never compromise architecture for convenience.
2. Never hardcode business logic.
3. Prefer plugins over monolithic code.
4. Every module must be independently testable.
5. Planner always decides before execution.
6. Specialized plugins override generic search.
7. Every tool execution must be verified.
8. Optimize for laptop hardware.
9. Maintain clear documentation.
10. This repository is the single source of truth for FRIDAY.
